# Wordle Maybe - GitHub Pages

This is a static build of the Wordle Maybe game built with React. Hosted on GitHub Pages.

## Live Demo

🔗 [Wordle Maybe Live](https://kirtikayush.github.io/wordle/)

## About

Wordle Maybe is a clone of the popular word guessing game where players have 6 tries to guess a 5-letter word.

## How to Play

- Enter your guesses using your keyboard.
- Green tiles mean the letter is correct and in the correct position.
- Yellow tiles mean the letter exists in the word but in a different position.
- Gray tiles mean the letter does not exist in the word.

## Tech Stack

- React
- CSS
- GitHub Pages (for deployment)

## Deployment

This is a static site served from GitHub Pages. To deploy:

1. Build the project

   ```bash
   npm run build
   ```

2. Push the contents of the `dist/` folder to the GitHub Pages repository.
